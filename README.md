# portugol-portfolio-projects
This repository contains a collection of projects and exercises developed using Portugol.
The goal is to practice fundamental programming concepts such as logic, algorithms, and problem-solving.
These projects represent my learning journey in programming and help strengthen the foundation needed for more advanced languages.

!!! The projects included here demonstrate real-world applications and continuous learning in Portugol development. Some of these programs may be broken. I created them when I was in 10th grade, and I still had a very basic and standard understanding of the IT world. But now that I'm in 13th grade and have much more knowledge, I intend to improve them all until they reach a perfect level.

!!! Please note, I am a native Portuguese speaker, so my programs will need to be translated for better understanding.
